# luftdaten_new

A new map for the Luftdaten project (OK Lab Stuttgart, Germany): https://luftdaten.info/

This map will show the PM10, the PM2.5, Temperature, Humidity, Pressure and AQI US.

It uses leaflet.js and D3.js.
